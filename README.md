## Jiam_Floaty
#해양 인명구조 디바이스
1. 개발 이유
- 바다에서 조난을 당했을 시 조난자의 위치를 모르기 때문에 구조가 힘들다.
- 그래서 사람마다 Floaty를 들고 있다면 구조자가 조난자의 위치를 알게 되고 구조 확률이 높아진다

2. 시나리오
- 난파된 배가 구조신호를 보낸다
- 모든 사람들은 floaty를 몸에 지니고 구명정 또는 바다로 뛰어든다.
- 구조대가 floaty 수신기를 들고 난파된 배의 구조신호가 발생한 곳으로 간다.
- 난파된 배를 중심으로 수색을 시작한다.
- 이 때 floaty수신기를 사람마다 들고 있다면 수신기가 자신의 위치정보를 송신하기 때문에 구조될 확률이 높아진다.

3. Floaty의 작동원리
- GPS모듈로 아두이노 자신의 위치를 저장한다
- GPS에서 저장한 정보를 LoRa 송신기가 LoRa 수신기로 송신한다(LoRa 송수신기간의 수신거리는 약 10km정도된다)
- GPS정보를 토대로 조난자의 위치를 구조대가 신속히 발견한다
